# Presentation Image Pack

These presentation assets are safe to include in talks, posters, lab meetings,
and project documentation. They contain no private research data.

## Files

- `hmm_discovery_workflow_overview.svg`: full research workflow from input to
  export.
- `hmm_discovery_no_code_user_journey.svg`: shows that normal users run the app
  by clicking through the interface rather than writing code.
- `hmm_discovery_database_strategy.svg`: explains discovery databases,
  VOGDB/Pfam annotation, scale databases, and specificity controls.
- `hmm_discovery_outputs_reproducibility.svg`: summarizes report tables,
  methods text, reproducibility JSON, and export ZIP contents.

PNG exports are generated from these SVG files when available. Use SVG for
crisp scalable slides and PNG for simple drag-and-drop into presentation tools.
